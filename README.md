Start a new app:

```sh
$ python manage.py startapp users
$ tree users/
users/
├── admin.py
├── apps.py
├── __init__.py
├── migrations
│   └── __init__.py
├── models.py
├── tests.py
└── views.py

2 directories, 7 files
```

Activate the virtual env:

```bash
source ll_env/bin/activate
```

Start the server:

```bash
python manage.py runserver
```
